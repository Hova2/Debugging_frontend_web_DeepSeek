Sobre este manual

🎯 Objetivo

Este manual nace de la necesidad de tener un recurso en español, práctico y estructurado para aprender debugging web desde cero. Está diseñado para principiantes absolutos que quieren entender no solo cómo arreglar errores, sino cómo pensar como un debugger.

📚 Estructura

El manual está organizado en 10 capítulos, cada uno con:

  • Explicaciones teóricas con ejemplos reales
  • Ejercicios prácticos para afianzar conceptos
  • Resúmenes y puntos clave
  • Enlaces a recursos adicionales

👨‍💻 Autor

Este manual es un proyecto vivo, creado para la comunidad de desarrolladores hispanohablantes. Si tienes sugerencias, correcciones o ideas, contribuye o abre un issue en el repositorio.
