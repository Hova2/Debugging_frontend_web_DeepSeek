2. Panel Elements El taller de HTML y CSS en vivo

El panel Elements (o "Inspector") es tu taller de reparación rápida. Aquí puedes ver la estructura HTML de cualquier página, modificar estilos CSS en tiempo real y experimentar sin miedo a romper nada porque los cambios son locales y temporales.

🔧 Analogía: Piensa en el panel Elements como un quirófano donde puedes operar una página web viva, ver sus huesos (HTML), su piel (CSS) y hacer pruebas antes de escribir el código definitivo.

2.1 Abrir y navegar el panel

Formas de abrirlo:

  • Atajo de teclado: F12 o Ctrl+Shift+C (Windows/Linux) / Cmd+Shift+C (Mac)
  • Menú contextual: Clic derecho en cualquier elemento → "Inspeccionar"
  • Menú del navegador: Más herramientas → Herramientas de desarrollo

Estructura del panel:

El panel Elements tiene dos partes principales:

  • Árbol DOM (izquierda): Muestra la estructura HTML de la página. Puedes expandir/colapsar etiquetas.
  • Panel de estilos (derecha): Muestra las reglas CSS aplicadas al elemento seleccionado, con su origen (archivo .css, etiqueta style, estilos en línea).

⚡ Ejercicio 2.1: Tu primera inspección

1. Abre el panel Elements ahora mismo (F12 o clic derecho → Inspeccionar).

2. Encuentra este mismo párrafo en el árbol DOM.

3. ¿Puedes ver mis atributos? Busca el class="exercise-box".

2.2 Modificar HTML en tiempo real

Una de las características más poderosas: puedes cambiar el texto, atributos y estructura HTML y ver los resultados al instante.

✏️ Editar texto

Doble clic en cualquier texto dentro del árbol DOM y escribe lo que quieras.

🏷️ Editar atributos

Doble clic en un atributo (como class="...") y cámbialo.

➕ Añadir elementos

Clic derecho en un nodo → "Edit as HTML" para escribir HTML nuevo, o "Copy" → "Copy element" para duplicar.

Prueba con este elemento:

¡Modifícame!
(desde el inspector)

⚡ Ejercicio 2.2: Manos al código

  1. Inspecciona la caja de arriba (la que dice "¡Modifícame!").
  2. Cambia el texto por "¡Lo logré!".
  3. Cambia el atributo class="demo-box" por class="demo-box modificado". ¿Notas algún cambio? (Si no, es porque no hay estilos para .modificado, pero el atributo cambió).
  4. Añade un atributo style="background: red;" directamente en la etiqueta HTML desde el inspector. La caja debería volverse roja.

💡 Todo cambio es temporal. Si recargas la página, vuelve al original.

2.3 El Box Model y panel de estilos

En la pestaña "Styles" (junto al árbol DOM) ves todas las reglas CSS aplicadas al elemento seleccionado. Pero hay una joya escondida: la pestaña "Computed" y el diagrama del Box Model.

El Box Model

Todo elemento HTML es una caja compuesta por:

  • Content: El contenido (texto, imagen)
  • Padding: Espacio interno, alrededor del contenido
  • Border: El borde
  • Margin: Espacio externo, separa del resto de elementos

En el panel Elements, abajo a la derecha, hay un diagrama interactivo que muestra estas dimensiones para el elemento seleccionado. ¡Puedes hacer clic en los números y modificarlos!

Caja de demostración

⚡ Ejercicio 2.3: Jugando con el box model

  1. Inspecciona la caja naranja de arriba (la que dice "Caja de demostración").
  2. En la pestaña "Styles", busca el diagrama del Box Model (suele estar al final, o en la pestaña "Computed").
  3. Haz clic en el número del padding (actualmente 2rem) y cámbialo a 3rem. Observa cómo se agranda la caja por dentro.
  4. Cambia el margin a 3rem también. ¿Ves más espacio alrededor?
  5. Ahora, desde la pestaña "Styles", desmarca temporalmente la propiedad background. La caja pierde su color. Vuelve a marcarla para restaurarlo.

💡 Desmarcar propiedades es una forma rápida de ver cómo afectan al diseño sin borrarlas permanentemente.

2.4 Forzar estados (:hover, :active, etc.)

¿Alguna vez quisiste inspeccionar un elemento que solo aparece al hacer hover o al enfocar? El panel Elements te permite "forzar" esos estados.

Cómo hacerlo:

  1. Selecciona el elemento en el árbol DOM.
  2. En la pestaña "Styles", busca el ícono :hov (un círculo con una línea punteada).
  3. Marca la pseudoclase que quieras forzar (:hover, :active, :focus, :visited, etc.)
  4. ¡El elemento se queda en ese estado aunque no estés interactuando!

⚡ Ejercicio 2.4: Atrapa ese hover

  1. Inspecciona el botón de arriba.
  2. En la pestaña "Styles", busca el botón :hov y marca la opción ":hover".
  3. Ahora el botón se queda en estado hover permanentemente, aunque no tengas el ratón encima.
  4. Observa los estilos que cambian (color de fondo, texto, transformación).
  5. Desmarca :hover para volver a la normalidad.

📌 Resumen del Capítulo 2

  • El panel Elements muestra el árbol DOM y los estilos aplicados.
  • Puedes editar HTML y CSS en tiempo real (cambios temporales).
  • El Box Model (margin, border, padding, content) es clave para entender el diseño.
  • Puedes forzar pseudoclases como :hover para inspeccionar estados.
  • El inspector es la herramienta más rápida para probar cambios visuales.
